        public static JDTPackage init()
        Creates, registers, and initializes the Package for this model, and for any others upon which it depends.

        This method is used to initialize JDTPackage.eINSTANCE when that field is accessed. Clients should not invoke it directly. Instead, they should simply access that field to obtain the package.
